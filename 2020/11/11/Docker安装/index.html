Docker简介

docker是一个开源的应用容器引擎，可以让我们快速的搭建一个理想的系统环境。对于如何在Centos上安装docker，官网上一共给出了三种安装方式：通过包管理器安装、通过rpm包安装以及使用给出的自动安装脚本安装。本文主要描述的是第一种通过包管理器安装。

Docker安装

删除旧版本的docker

  • 如果之前你的机器并未安装过docker可忽略此步骤

使用yum list installed | grep docker-*可以查看是否安装有docker的包，如有安装便可以使用yum remove将其删掉，也可以直接使用官方文档上的命令不管是否安装直接尝试卸载

配置相关工具

安装必须的工具包：yum-uitl、device-mapper-persistent-data以及lvm2

添加docker的库

打开其他版本docker的支持

安装docker

  • 执行yum install docker-ce安装
  • 使用systemctl start docker运行docker的服务
  • 执行docker run hello-world测试docker是否能正常使用

至此安装完毕
