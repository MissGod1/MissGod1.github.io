  • 发表时间: 2020
  • 主题: webshell检测
  • 关键词: 机器学习, n-gram, TF-IDF, 贝叶斯, webshell, VLD
  • 数据集: 来自于github上收集的webshell和开源的php项目
  • 流程图

评价标准

True Class Predicted Result Predicted Result
– Webshell Normal Page
Webshell True Positive(TP) False Negative(FN)
Normal Page False Positive(FP) True Negative(TN)
  • TP: 将Webshell文件识别为Webshell文件
  • FN: 将Webshell文件识别为正常文件
  • FP: 将正常文件识别为Webshell文件
  • TN: 将正常文件识别为正常文件
  • 精准度: Precision=TPTP+FPPrecision=\frac{TP}{TP+FP}Precision=TP+FPTP​
  • 准确度: Accuarcy=TP+TNTP+TN+FP+FNAccuarcy=\frac{TP+TN}{TP+TN+FP+FN}Accuarcy=TP+TN+FP+FNTP+TN​
  • 召回率: Recall=TPTP+FNRecall=\frac{TP}{TP+FN}Recall=TP+FNTP​
  • F1: F1=2∗Precision∗RecallPrecision+RecallF1=\frac{2*Precision*Recall}{Precision+Recall}F1=Precision+Recall2∗Precision∗Recall​

数据处理

  1. 使用VLD将PHP文件转换为操作码序列
  2. 将获得的操作码序列使用2-gram进行预处理
  3. 将2-gram处理后的数据再使用TF-IDF进行处理
  4. 使用贝叶斯模型训练处理后得到的数据
